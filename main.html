Overview

I was inspired to create this project when the powerball lottery was over a billion dollars. My sister and I got into a discussion about how it seems like there is a way to space your numbers in such a way that you are guarenteed to win. This is my attempt to prove this theory right.

Accuracy By Volume

This option generates a list of tickets. Each possible number is distributed evenly between all tickets.

Set Outcome

This option uses the same winning number for all tickets. Generates new tickets each time

Purely Random

This generates new random tickets, and a new winning number each time.

Conclusion

In conclusion, this experiment was ultimately a failure.
